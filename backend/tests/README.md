# Tests del Sistema de Auto-Detección de Esquemas

Esta carpeta contiene los scripts de prueba para el sistema de auto-detección de esquemas.

## 📁 Archivos de Prueba


### 🧪 `test_schema_detection.js`
**Propósito:** Prueba completa del sistema de auto-detección de esquemas.

**Función:**
- Inicializa la base de datos con migraciones
- Prueba la detección de esquemas desde SQLite
- Verifica que TableSchema funcione con auto-detección
- Prueba la creación de modelos con el nuevo sistema
- Ejecuta validaciones con esquemas detectados
- Muestra información completa del sistema

**Cuándo usarlo:**
- Para pruebas integrales del sistema
- Para verificar que todo funciona junto
- En desarrollo y testing
